# Benchmark Results

Results are generated by running:

```bash
make benchmark
```

This directory contains:

- `backend_comparison.json` — raw latency/throughput data per backend/concurrency/batch
- `backend_comparison.md` — Markdown summary table generated by `analyze_results.py`
- `tensorrt_build.log` — TensorRT engine build log from `make build-tensorrt`

Results are excluded from git (see `.gitignore`). To reproduce, run the benchmark commands from the repo root.
